# **App Name**: OM Suivi

## Core Features:

- Profile Management: Allow users to enter and manage contractual information such as base hours and overtime rates.
- Manual Time Tracking: Integrate a manual time tracking system for logging work hours.
- Geolocation Integration: Use the tool for automatic geolocation to ensure time entries are accurately recorded. Prompt user for approval before recording the entry
- Customizable Reminders: Set customizable reminder alerts to prevent missed time entries.
- Custom Report Generation: Generate daily, weekly, and monthly reports for tracking hours worked and overtime.
- Team Data Sharing: Implement data sharing functionality to share and summarize hours worked among colleagues, as permissioned.

## Style Guidelines:

- Primary color: Dark blue (#223843) for professionalism and reliability, reflecting a serious approach to time tracking.
- Background color: Light gray (#E8EBE4), a desaturated hue of the primary color, providing a neutral and clean backdrop.
- Accent color: Teal (#39A9DB) to highlight key interactive elements and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif font.
- Use clean, simple icons to represent different functions (time tracking, reports, settings, team, profile) within the app.
- Maintain a clean, organized layout with clear sections for each function: Profile, Time Tracking, Reports, and Team. Prioritize a user-friendly design to make the app easy to navigate.
- Use subtle animations and transitions to provide feedback on user interactions and enhance the app's overall user experience. Avoid excessive animations that could distract from the app's functionality.
